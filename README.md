# Dr. Omar Portfolio

A modern, premium business portfolio built with Next.js, React, and Tailwind CSS v4.

## Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application running.

## Project Structure
- `src/app/` - Next.js App Router for all page routing.
- `src/components/layout/` - Global components like Navbar and Footer.
- `src/components/home/` - Modular components for the homepage (Hero, About, etc.).
- `src/components/program[X]/` - Components dedicated to specific business programmes.

## Tech Stack
- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Language**: TypeScript
